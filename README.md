# Adventure-Works Sales Insights Dashboard  

### Developed by **[Anand Cinenkanolu]**

This repository features a Power BI project built on the AdventureWorks dataset, designed to break down sales performance across products, customers, and regions. The dashboard uses clean modeling, curated DAX measures, and interactive visuals to turn raw records into meaningful insights.

## ğŸ¯ Project Goal  

AdventureWorks includes a wide range of transactional data, but making sense of it requires structure.  
This dashboard helps users:

- Track revenue and profit changes  
- Compare product category performance  
- Analyze customer segments and revenue contribution  
- Understand geographic differences in sales  
- Explore detailed product-level behavior  

The end result is a clear and intuitive view of overall business performance.
